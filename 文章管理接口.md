
1. 查看文章分页列表a
接口: `/api/admin/platform/article/list`  
请求方式: GET  
请求参数:
{
  "cid": 1,           // 分类id（可选）
  "type": 0,          // 文章类型（可选，0-动态，1-咨询）
  "title": "标题",     // 文章标题（可选）
  "author": "作者",    // 文章作者（可选）
  "page": 1,          // 页码
  "limit": 10         // 每页数量
}

{
  "code": 200,
  "message": null,
  "data": {

        "id": 1,

2. 新增文章
接口: `/api/admin/platform/article/save`  
请求方式: POST  
请求参数:
{
  "cid": 1,                              // 文章分类id（必填）
  "type": 0,                             // 文章类型（必填，0-动态，1-咨询）
  "title": "文章标题",                     // 文章标题（必填，最多200字符）
  "author": "作者",                       // 文章作者（必填，最多50字符）
  "cover": "https://example.com/cover.jpg", // 文章封面（必填，最多255字符）
  "synopsis": "文章简介",                  // 文章简介（必填，最多200字符）
  "content": "文章内容",                   // 文章内容（必填）
  "isHot": false,                        // 是否热门（必填）
  "sort": 1                              // 排序（可选，范围0-9999）
}
响应结果:
{
  "code": 200,
  "message": null,
  "data": "操作成功"
}
